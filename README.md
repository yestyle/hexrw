# hexrw

A CLI utility to read / write hexadecimal value from / to a file.

# Install

Use the following command to install `hexrw` executable to `~/.cargo/bin` directory:

```
cargo install hexrw
```

# Usage

```
hexrw [-w <write>] [-r <read>] [-d <delay>] <file>
```
